    # 从开通账号到直播带货

    流程

    注册信息—货品广场—意向库—直播招商—直播中心

    # 首先 完成账号注册

    # 一、完成注册

    # 账号主体选择—MCN机构

    # 1.1 红人信息绑定

    至少拥有一名主播以上，可在 直播排班 中指定主播开播\带货

    # 二、货品广场

    选择意向带货商品

    # 2.1 当你完成注册后，跳转首页

    # 2.2 添加意向商品

    直接点击加入意向库

    #

    分享单以及商品链接功能，请查看「高级功能-关于分享单」

    # 2.3 意向库管理

    # 意向库内包含你所有的初选货品

    #

    账户保证金额度以及使用情况 点击查询

    # 2.4 申请样品

    # 确认样品的规格和数量

    # 2.5 填写申请信息

    自动填写默认地址，信息填写错误或需要更改，请查看下面 样品管理 相关内容介绍

    # 三、样品管理

    # 3.1 寄样申请管理

    处于 待审核/待收款/需寄回 的样品在这里完成统一的核对和物流管理

    # 3.2 交易记录和保证金充值

    用来查询账号资金使用情况，查看详细说明请点击「交易保证金」

    当保证金不足或者是部分品牌设置，需要成本购样，请根据需要，对保证金进行充值。

    # 四、直播招商/自有商品

    支持外部带货商品链接导入\平台招商对接

    如有下列需求

    • 头部主播货品匹配发起

    • 解决入驻品牌且未入库商品的寻求

    • 厂家品牌佣金和专场宣传费用有沟通需求

    # 4.1 招商信息发布与填写

    # 4.2 招商完成

    招商更多细节请咨询客服。

    # 五、直播中心

    # 5.1 创建直播入口

    # 5.2 添加直播信息

    通过直播中心-直播排版-添加直播场次 可指定绑定主播\外部合作主播进行开播

    # 5.3 直播组货商品选择

    通过入口 直播中心-直播排版-直播组货 货品可以从意向库、主播招商、自有商品三个地方添加 在您添加完成商品后，可以用过配合导出 组货单: 以Excel的形式自动下载,线下打印并核对带货货品！

    # 5.4 手卡\背景设置

    • 配合直播助手一体机使用 「直播助手指南」 我们大部分商品提供手卡信息以及默认背景，如有需要可行修改。

    # 5.5 开始直播

    直接点击 开始直播 完成开播 或是按照计划时间准时开播

    直播过程中随时可以查看带货动态，结束自动生成本场数据

    进行到这里已经基本完成了页面设置全部的操作

    直播过程中配合请搭配硬件直播助手使用

    关于硬件助手，我们有专门的介绍，请点击 「直播助手指南」，查看使用详情

    目前平台支持以下功能：

    · 拥有排行榜、热门视频、脚本库、直播电商等数据分析和查询功能

    · 提供热门视频、直播、音乐、爆款商品及优质账号；

    · 提供全方位数据支持，曾为多位明星提供直播场控支持；

    · 实现多账号运营分析，立体化展现直播、视频、商品等数据，实时秒级监测；

    · 利用大数据追踪直播和短视频市场趋势及流量趋向；

    · 助力创作者账号运营内容定位、粉丝增长、粉丝画像优化及流量变现。

    MCN机构

    服务运营公司需要分析的竞品数据，挖掘黑马主播等带货渠道拓宽，帮助公司网红孵化平台建立及个体达人培养
